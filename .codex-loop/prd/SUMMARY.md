# PRD 요약

생활권 기반 시니어 안심주행 특약의 AI 모델 견본과 제출용 시각화 자료를 만든다.

핵심 구조는 다음과 같다.

```text
Trip 데이터
-> 생활권 생성 AI
-> 운전행동 feature
-> 평소패턴 변화 감지 AI
-> 점수 계산
-> 추가 리워드 / 기본 유지 / 예방 케어 판단
-> 시각화 자료와 제안서 source
```

AI는 사고를 직접 예측하지 않는다. 고객별 생활권과 평소 운전패턴을 학습하고, 최근 운전이 평소와 달라졌는지 감지한다.

TAAS 또는 공공 사고통계는 개인 예측 라벨이 아니라 과속, 안전거리 미확보, 야간, 노인운전자 사고 등 위험행동 가중치 근거로만 활용한다.

제출 자료는 PDF부터 만들지 않는다. Markdown source, CSV, Python script, SVG figure를 먼저 만들고 Ralph review/render gate로 검토한다.
