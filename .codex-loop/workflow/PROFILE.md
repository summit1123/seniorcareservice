# Summit 워크플로우 프로필

프로필: proposal-only
프로필-라벨: 제안서 전용
기본-모드: proposal

## 이런 경우에 사용합니다
- 공모전, 제안서, 사업계획서, 심사자료가 핵심 산출물일 때
- 구현보다 reviewer-facing narrative와 증거 구조가 먼저일 때

## 목표 산출물
- 폼 답변 초안
- 심사위원용 첨부 문서 또는 PDF 원고
- 근거 목록과 검토 체크리스트

## 단계 맵
| 단계 ID | 권장 모드 | 완료 결과 |
| --- | --- | --- |
| onboarding | proposal | 자료, 목표, 심사 맥락을 정리합니다. |
