# Summit 워크플로우 상태

프로필: proposal-only
목표: 생활권 기반 시니어 안심주행 특약의 AI 모델 견본과 제출용 시각화 자료를 만든다
현재-단계: onboarding
현재-모드: proposal
상태: 진행 중

## 단계 체크리스트
- [> ] onboarding: 온보딩 (proposal)
- [ ] idea-screening: 인사이트 및 아이디어 선별 (proposal)
- [ ] deep-research: 딥리서치 (proposal)
- [ ] proposal-package: 제출 패키지 작성 (proposal)
- [ ] review-and-submit: 최종 검토 및 제출 (proposal)

## 현재 단계 완료 결과
- 자료, 목표, 심사 맥락을 정리합니다.

## 다음 두 단계
- idea-screening (proposal): 여러 방향을 비교하고 이번에 밀 방향을 하나 고릅니다.
- deep-research (proposal): 선택한 방향의 근거와 자료를 잠급니다.

## 다음 단계로 넘기는 법
- 현재 단계가 정말 완료되었을 때 이 파일과 config를 함께 업데이트합니다.
- `python3 scripts/summit_start.py advance --stage <stage-id>`로 다음 단계로 이동합니다.
