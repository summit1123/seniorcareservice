# Summit 워크플로우 온보딩

프로필: proposal-only
프로필-라벨: 제안서 전용
목표: 생활권 기반 시니어 안심주행 특약의 AI 모델 견본과 제출용 시각화 자료를 만든다
상태: 초안

이 문서는 `/ralph-start` 이후 가장 먼저 잠그는 상위 온보딩 문서입니다. 첫 질문은 사용자가 지금 무엇을 하고 싶은지 확인하는 것입니다.

- 대기 중.

- 대기 중.
